API Reference#

ImageViewer#

PropTypeDefaultDescription
imagesstring | string[]—One or more image URLs to display.
openboolean—Controls whether the viewer is open.
onClose() => void—Called when the viewer is dismissed.
indexnumber—Controlled image index. Use with onIndexChange.
initialIndexnumber0Index of the image to show first.
onIndexChange(index: number) => void—Called when navigation changes the active image.
altstring"Image"Alt text prefix applied to each image.
containerElement | DocumentFragment | nulldocument.bodyPortal target. Pass null or a custom element to control where the viewer renders.
classNamestring—Additional CSS class for the fullscreen overlay.
toolbarClassNamestring—Additional CSS class for the toolbar.
thumbnailsClassNamestring—Additional CSS class for the thumbnails strip.
