## Usage

```tsx
import { ImageViewer } from "@/components/pro/pro-viewer/image"
import { useState } from "react"

export default function Example() {
  const [open, setOpen] = useState(false)

  return (
    <>
      <button onClick={() => setOpen(true)}>Open</button>
      <ImageViewer
        images={["https://example.com/photo.jpg"]}
        open={open}
        onClose={() => setOpen(false)}
      />
    </>
  )
}
```

## API Reference

### ImageViewer

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `images` | `string \| string[]` | — | One or more image URLs to display. |
| `open` | `boolean` | — | Controls whether the viewer is open. |
| `onClose` | `() => void` | — | Called when the viewer is dismissed. |
| `index` | `number` | — | Controlled image index. Use with `onIndexChange`. |
| `initialIndex` | `number` | `0` | Index of the image to show first. |
| `onIndexChange` | `(index: number) => void` | — | Called when navigation changes the active image. |
| `alt` | `string` | `"Image"` | Alt text prefix applied to each image. |
| `container` | `Element \| DocumentFragment \| null` | `document.body` | Portal target. Pass `null` or a custom element to control where the viewer renders. |
| `className` | `string` | — | Additional CSS class for the fullscreen overlay. |
| `toolbarClassName` | `string` | — | Additional CSS class for the toolbar. |
| `thumbnailsClassName` | `string` | — | Additional CSS class for the thumbnails strip. |

## Keyboard Shortcuts

| Key | Action |
| --- | --- |
| `←` / `→` | Previous / Next image |
| `Esc` | Close viewer |
| `Scroll` | Zoom in / out |
